## Setup
>Install Front-End Dependencies
```
npm run frontend:install
```
>Install API Dependencies
```
npm run api:install
```
>Front-End Build
```
npm run frontend:build
```
>API Build
```
npm run api:build
```
> Deploy Frontend
```
npm run frontend:deploy
```
>Deploy API
```
npm run api:deploy
```
